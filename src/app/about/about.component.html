About this Application

In this particular application you are required to enter your github username and some information will be displayed

The information displayed will include:

  1. Your profile photo
  2. Your login name
  3. The date you created your account
  4. The number of public repositories present
  5. A link to your profile

The search results will also inlude repository details

  1. Languages used
  2. A clone repository url
  3. The name of the Repo
  4. The date of creation of the repo
